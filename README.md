## 📌 Sobre o Projeto

Uma aplicação web interativa que ensina o conceito de **objetos em JavaScript** através da criação de um **Smartphone** como objeto principal. O projeto explora propriedades, métodos, manipulação do DOM e reflexões sobre Inteligência Artificial no dia a dia.

### 🎯 Objetivos de Aprendizado

- ✅ Criar objetos JavaScript com propriedades e métodos
- ✅ Utilizar funções para dividir problemas complexos
- ✅ Aplicar métodos para manipular o DOM
- ✅ Refletir sobre o uso de IA em tarefas diárias
- ✅ Entender a relação entre objetos e Inteligência Artificial
